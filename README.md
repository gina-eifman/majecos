# MajeCos — интернет-магазин уходовой косметики

Учебный проект — главная страница интернет-магазина с акцентом на визуальную составляющую и интерактивные элементы.


## Обзор
Проект представляет собой одностраничный сайт с каруселями товаров, лентой рилсов и адаптивной сеткой категорий. Основная задача — реализовать плавные анимации и циклическую карусель для видео-рилсов.

🔗 **Демо:** проект можно увидеть по ссылке: [https://gina-eifman.github.io/majecos/](https://gina-eifman.github.io/majecos/).


## Технологии
1. HTML5: Семантическая разметка страницы.
2. CSS3: Стилизация, анимации, адаптивная вёрстка (Flexbox, Grid).
3. JavaScript (ES6+): Логика каруселей, динамический рендер, анимации.
4. БЭМ: Методология именования классов.

## Реализованные функции

### 1. Карусель баннеров
- автоматическое перелистывание (каждые 5 секунд);
- ручное управление стрелками;
- индикаторы (dots) с активным состоянием;
- пауза при наведении курсора на карусель.

### 2. Циклическая карусель рилсов
Главная техническая сложность проекта. Карусель работает по следующему принципу:
- пользователь видит несколько карточек **одновременно**;
- при нажатии на стрелку карточки **циклически сдвигаются** — после последнего рилса снова появляется первый;
- масштабирование карточек относительно центра:
  - центральная: `scale(1.1)`;
  - ближние (слева/справа): `scale(0.9)`;
  - дальние (слева/справа): `scale(0.8)`.

**Алгоритм карусели рилсов:**
1. Хранение порядка отображения в массиве `visibleOrder` из 5 индексов.
2. При клике на стрелку все индексы сдвигаются на +1 или -1 (с зацикливанием через `% totalCards`).
3. Рендер новых карточек с небольшой анимацией затухания для плавности.
4. Блокировка повторных кликов во время анимации (`isAnimating`).

**Пример сдвига индексов**
```javascript
if (direction === 'next') {
    visibleOrder = visibleOrder.map(idx => (idx + 1) % totalCards);
} else {
    visibleOrder = visibleOrder.map(idx => (idx - 1 + totalCards) % totalCards);
}
```
### 3. Сетка категорий с фиксированными размерами
Использование CSS Grid с явно заданными областями:
```css
text
.grid {
    grid-template-areas: 
        "face lips hair"
        "face hands hands";
}
```
На мобильных устройствах сетка перестраивается в вертикальный список.

### 4. Динамический рендер контента
Категории и список уходовой косметики генерируются JavaScript из массивов данных. Это позволяет легко добавлять или изменять позиции без правки HTML.

## Адаптивность
Проект корректно отображается на:
- десктопах (>1200px) — полная сетка категорий;
- планшетах (768px–1200px) — уменьшенные отступы, перестроение категорий;
- мобильных устройствах (<768px) — вертикальная сетка, центрирование элементов.

## Отступления от макета
В процессе разработки были приняты следующие решения, отличающиеся от исходного макета:
- анимация переключения рилсов — вместо горизонтального сдвига реализовано лёгкое затухание, что обеспечило стабильную работу циклической карусели без технических артефактов;
- цветовая гамма — исходные яркие тона были заменены на более пастельные оттенки розового для лучшего визуального восприятия;
- способ управления рилсами — стрелки карусели размещены по бокам, а не снизу, для единообразия с баннером.
